Settings page — how should guardrail modes be controlled?

Each detector category (Secrets, PII, Prompt Injection) has four modes: off · alert · block · redact

A

Segmented button row (inline)

Modes shown as a 4-button toggle per row. Fast to scan and change — no dropdowns to open.

Dropdown select per row

Each category has a <select> dropdown. Compact, familiar — but requires a click-to-open interaction per setting.

Cards with pill selector + description

Each detector category gets a card. Mode is selected with buttons, and the card shows a brief description of what that mode does. More space but clearer for new users.
