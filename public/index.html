    Welcome — let's get the proxy configured

    AIRelay hasn't been pointed at an upstream API yet. Pick a provider below and we'll generate the .env file you need.

    The proxy never sees or stores your API key — your SDK sends it on each request and the proxy forwards it untouched. You only configure the upstream URL here.

    https://api.anthropic.com

    Full URL including scheme. Use http:// for plain HTTP upstreams.

    SDK baseURL ends with this.

    HTTP port the dashboard + proxy listen on.

    Informational only — what your team uses to reach this box. Set DNS / Tailscale / hosts file separately. See CONFIGURATION.md for DNS setup.
